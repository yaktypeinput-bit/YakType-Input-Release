# 初始化与引导流程 (Onboarding)

## 概要说明

本文档详述了 YakType 的“冷启动”初始化流程。该流程旨在通过引导式阶梯（Wizard）确保用户在首次使用时正确配置系统权限与首个听写引擎。该设计遵循“净地原则 (Blank Slate)”，即系统不会在未获得用户确认的情况下默认开启任何后台引擎。

---

## 引导流程阶段 (The 4-Step Wizard)

初始化过程被划分为四个严格定义的阶段，以确保交互的线性与配置的完备性。

### 阶段 1：系统权限申请 (Permissions)
*   **目标**：获取应用运行的核心权限。
*   **关键权限项**：
    *   **麦克风 (Microphone)**：采集语音输入。
    *   **辅助功能 (Accessibility)**：**核心权限**。驱动全局快捷键监听，并利用 `AXUIElement` 接口实现文本自动填充至目标 App 的输入焦点。
    *   **语音识别 (Speech Recognition)**：仅在用户选择 **Apple (原生)** 引擎执行 ASR 任务时强制要求，由系统的 `SFSpeechRecognizer` 组件驱动。
    *   **屏幕录制 (Screen Recording)**：仅在用户将 `AudioSource` 切换为“系统音频”以捕获非本应用发出的声学信号时触发。
*   **交互逻辑**：提供“状态刷新”按钮，利用 `PermissionManager` 轮询各权限的授权状态。

### 阶段 2：听写引擎选型 (Engine Selection)
*   **目标**：确定首个 `ProcessingPipeline` 的听写组件。
*   **可选引擎方案**：
    *   **Apple**：基于 `On-Device` 机器学习模型，支持完全离线识别，无网络延迟。
    *   **Gemini / OpenAI**：云端大模型驱动，具备极高的语义纠错能力。
    *   **阿里云 (通义千问)**：针对中文 ASR 深度优化的云端引擎。
*   **配置建议**：若追求极致响应度且网络受限，应优先选择 Apple 引擎。

### 阶段 3：参数持久化 (Configuration)
*   **目标**：配置 API 密钥及 VAD（语音活动检测）参数。
*   **关键参数说明**：
    *   **VAD 阈值 (Silence Threshold)**：默认为 1800ms。音信号低于能量阈值持续此时长后，系统将判定为句子结束，自动截断音频流并提交转录。
*   **设计模式**：采用紧凑型 Label-on-Top 布局，实时同步变更至 SwiftData 存储层。

### 阶段 4：完成设置 (Completion)
*   **目标**：固化配置并关闭引导。
*   **副作用**：设置 `onboardingCompleted` 标志为 true，并根据初始化选择生成的首个 `EngineProfile` 指纹。

## 运行时 UI 状态维护 (Floating HUD)

初始化完成后，用户的每一次语音交互均由 **Floating HUD (状态悬浮窗)** 进行实时反馈。

| 状态 | 交互含义 | 备注 |
| :--- | :--- | :--- |
| **Recording** | 实时音频捕获中。 | 显示波形能量级与计时。 |
| **Transcribing** | ASR 处理阶段。 | 正在将音频切片转换为文本。 |
| **Polishing** | AI 润色阶段。 | 调用后处理引擎通过 Prompt 模板优化结果。 |
| **Success/Error** | 流程终结状态。 | 成功后执行自动填充，错误则显示具体 Code。 |

---

## 核心设计规范

### 1. 固定导航底座 (Fixed Navigation Footer)
*   **稳定性**：底座固定在 Modal 底部，确保交互热区的绝对稳定性。
*   **交互语义**：支持 `Esc` 键实时取消当前的初始化操作。

### 2. 净地原则逻辑
*   **触发机制**：`MigrationManager` 在启动时通过 `performMigrationAndConsistencyCheck` 检查是否存在任何 `EngineProfile`。若为空，强制激活引导流程。

## 常见异常处理

| 现象 | 可能原因 | 处理对策 |
| :--- | :--- | :--- |
| **按钮置灰无法下一步** | 核心权限（麦克风/辅助功能）未授予。 | 检查阶段 1 中的权限状态标签。 |
| **配置表单显示不全** | 屏幕纵向空间受限。 | 内容区支持垂直滑动。 |
| **返回后配置丢失** | 内存状态未及时同步。 | 系统已实现实时持久化，返回前内容已保存。 |
