# YakType Prompt Hub

## 概要说明

Prompt Hub 是 YakType 的核心提示词仓库，旨在为用户提供经过优化的 AI 润色与处理逻辑。这些提示词针对语音录入场景进行了特殊调优，以解决语音转写中常见的口水词、逻辑断续及书面语转换等问题。

## 提示词目录 (Directory)

| 提示词文件                            | 功能描述                         | 核心目标                                       |
| :------------------------------------ | :------------------------------- | :--------------------------------------------- |
| **[语音听写润色](dictation.md)**      | 处理原始语音转录文本             | 语义自愈、消除冗余、保留人称口吻、规范分段排版 |
| **[翻译润色 (中英)](translation.md)** | 中文至地道英语的翻译与翻译后润色 | 消除中式英语、语境自适应、直接输出纯净文本     |

## 使用指南

- **自定义修改**：用户可以克隆此仓库并根据自己的专业领域（如：法律、医疗、编程）调整特定文件的提示词细节。
- **引用逻辑**：YakType 客户端在执行 `Polishing Pipeline` 时，会加载这些 Markdown 文件作为核心指令集。
